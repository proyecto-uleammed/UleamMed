# UleamMed Mobile

Frontend movil basico en Flutter para la plataforma UleamMed.

## Pantallas incluidas

- Inicio con saludo, estado de animo y resumen del dia
- Medicamentos con horarios de ejemplo
- Citas medicas
- Actividades del dia
- Contactos de emergencia

## Requisitos

- Flutter SDK 3.12 o superior

## Ejecutar

```bash
cd mobile
flutter pub get
flutter run
```

## Notas

- Interfaz accesible con tipografia grande y paleta azul
- Datos de demostracion; integracion con backend pendiente
